Easy2Convert TGA to PNG
Version 1.4, Freeware
http://www.easy2convert.com/tga2png/

Introduction
------------
Easy2Convert TGA to PNG is a small freeware 
utility for converting Truevision Targa files (.tga, .targa) 
to Portable Network Graphics files (.png) easily and quickly.

System requirements
-------------------
 * Windows 2000/XP/2003/Vista/7/8
 * less than 3 Mb of free space on hard disk
 
Installation
------------
Run an installer/unzip all files to any folder and run 
the executable file (usually, "tga2png.exe").

Getting started
---------------
1. Select a tga-file to convert
2. Select an output png-file
3. Click on the "Convert" button.
